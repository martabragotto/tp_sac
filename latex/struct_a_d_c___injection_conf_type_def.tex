\doxysubsection{Detailed Description}
Structure definition of ADC group injected and ADC channel affected to ADC group injected. 

\begin{DoxyNote}{Note}
Parameters of this structure are shared within 2 scopes\+:
\begin{DoxyItemize}
\item Scope channel\+: Injected\+Channel, Injected\+Rank, Injected\+Sampling\+Time , Injected\+Single\+Diff, Injected\+Offset\+Number, Injected\+Offset, Injected\+Offset\+Sign, Injected\+Offset\+Saturation
\item Scope ADC group injected (affects all channels of injected group)\+: Injected\+Nbr\+Of\+Conversion, Injected\+Discontinuous\+Conv\+Mode, Auto\+Injected\+Conv, Queue\+Injected\+Context, External\+Trig\+Injec\+Conv, External\+Trig\+Injec\+Conv\+Edge, Injec\+Oversampling\+Mode, Injec\+Oversampling. 
\end{DoxyItemize}

The setting of these parameters by function HAL\+\_\+\+ADCEx\+\_\+\+Injected\+Config\+Channel() is conditioned to ADC state. ADC state can be either\+:
\begin{DoxyItemize}
\item For all parameters\+: ADC disabled (this is the only possible ADC state to modify parameter \textquotesingle{}Injected\+Single\+Diff\textquotesingle{})
\item For parameters \textquotesingle{}Injected\+Discontinuous\+Conv\+Mode\textquotesingle{}, \textquotesingle{}Queue\+Injected\+Context\textquotesingle{}, \textquotesingle{}Injec\+Oversampling\textquotesingle{}\+: ADC enabled without conversion on going on injected group.
\item For parameters \textquotesingle{}Injected\+Sampling\+Time\textquotesingle{}, \textquotesingle{}Injected\+Offset\textquotesingle{}, \textquotesingle{}Injected\+Offset\+Number\textquotesingle{}, \textquotesingle{}Injected\+Offset\+Sign\textquotesingle{}, \textquotesingle{}Injected\+Offset\+Saturation\textquotesingle{}, \textquotesingle{}Auto\+Injected\+Conv\textquotesingle{}\+: ADC enabled without conversion on going on regular and injected groups.
\item For parameters \textquotesingle{}Injected\+Channel\textquotesingle{}, \textquotesingle{}Injected\+Rank\textquotesingle{}, \textquotesingle{}Injected\+Nbr\+Of\+Conversion\textquotesingle{}, \textquotesingle{}External\+Trig\+Injec\+Conv\textquotesingle{}, \textquotesingle{}External\+Trig\+Injec\+Conv\+Edge\textquotesingle{}\+: ADC enabled and while conversion on going on ADC groups regular and injected. If ADC is not in the appropriate state to modify some parameters, these parameters setting is bypassed without error reporting (as it can be the expected behavior in case of intended action to update another parameter (which fulfills the ADC state condition) on the fly). 
\end{DoxyItemize}
\end{DoxyNote}

Specifies whether the context queue feature is enabled. This parameter can be set to ENABLE or DISABLE. If a new injected context is set when queue is full, error is triggered by interruption and through function \textquotesingle{}HAL\+\_\+\+ADCEx\+\_\+\+Injected\+Queue\+Overflow\+Callback\textquotesingle{}. Caution\+: This feature request that the sequence is fully configured before injected conversion start. Therefore, configure channels with as many calls to HAL\+\_\+\+ADCEx\+\_\+\+Injected\+Config\+Channel() as the \textquotesingle{}Injected\+Nbr\+Of\+Conversion\textquotesingle{} parameter.
